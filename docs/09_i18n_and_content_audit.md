# 09 多語系內容建置與內容審計 (i18n & Content Audit)

本文件摘要說明了雪場教練預約平台在國際化、內容呈現以及預約巫師系統上的最終建置成果。

## 1. 語系同步概況 (Syncing Status)
我們已將以下所有譯文檔案與 `zh_TW` (繁體中文) 主檔完成同步，解決了部分頁面切換語言時出現原始鍵值 (Literal Keys) 的問題。

| 語系碼 | 語言 | 主要特色 |
| :--- | :--- | :--- |
| `zh_TW` | 繁體中文 | 主導航、品牌故事與分階教學的標準基準。 |
| `zh_HK` | 廣東話 (HK) | **深度在地化**。採用道地粵語口吻（如「唔單止係」、「宜家預訂」）。 |
| `en` | 英語 (EN) | 採用國際化的專業滑雪術語（如「Japow」, 「Carving」）。 |
| `ja` | 日語 (JA) | 鎖定日本在地市場，採用正式且具吸引力的行銷文案。 |

## 2. 核心功能更新：預約巫師 (Booking Wizard)
預約流程已全面進化為「四階段引導」模式，顯著提升用戶完成率：

### A. 整合式雙層標題 (Unified Multi-tier Header)
*   **上層：** Logo 與全域語系切換。
*   **下層 (Stepper)：** 整合在 Header 內的進度追蹤條，支援歷史步驟回溯點擊。

### B. 四階段流程 (Phase 01-04)
1.  **選擇課程：** 磁貼式選擇，選定後自動摺疊並摘要顯示。
2.  **預約日期：** 互動式日曆，選定時段後自動推進。
3.  **教練選擇：** 加入教練個人化名片（Kenji / Mika），強化信任感。
4.  **結帳畫面：** 專業的訂單總結與多元支付方式選擇 (Credit Card / Line Pay)。

## 3. 視覺優化紀錄
*   **移除冗餘資訊：** 刪除了頁面中央多餘的 "Reservation Journey" 與 "預約課程" 標題，專注於引導巫師操作。
*   **響應式設計：** 步驟條在手機版會自動隱藏長文字標籤，僅保留節點與當前步驟縮寫，確保小螢幕不重疊。
*   **程式碼質量：** 修復了 TypeScript 類型定義問題，並優化了 Tailwind CSS v4 的類名寫法（如 `bg-white/2`）。
